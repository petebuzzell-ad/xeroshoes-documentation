Introduction

This enhanced table of contents provides a modern, user-friendly navigation experience for documentation sites. It automatically highlights the current section as users scroll through the content, making it easy to track their progress and navigate efficiently.

Try it out: Scroll through this page and watch how the table of contents highlights the current section automatically!

Features

The enhanced TOC includes several powerful features designed to improve the user experience:

Scroll Highlighting

Uses the Intersection Observer API to efficiently detect which section is currently in view and highlights the corresponding TOC item. This provides real-time feedback about the user's current position in the document.

Smooth Scrolling

Clicking on any TOC link smoothly scrolls to the target section, providing a polished navigation experience. The scrolling behavior is customizable and works across all modern browsers.

Performance Optimized

Built with performance in mind, using modern web APIs like Intersection Observer instead of scroll event listeners. This ensures smooth performance even on long documents with many sections.

Implementation

Adding the enhanced TOC to your documentation site is straightforward and requires minimal setup.

Basic Setup

To add the enhanced TOC to your site, simply include the JavaScript file and ensure your HTML structure follows the expected pattern:

<!-- Include the enhanced TOC script --> <script src="enhanced-toc.js"></script> <!-- Your HTML structure --> <nav class="toc-sidebar"> <h3>Table of Contents</h3> <ul> <li><a href="#section1">Section 1</a></li> <li><a href="#section2">Section 2</a></li> </ul> </nav> <main> <section id="section1">...</section> <section id="section2">...</section> </main>

Customization

The TOC can be customized through CSS variables and configuration options. You can modify colors, spacing, and behavior to match your site's design system.

API Reference

The enhanced TOC provides a public API for manual control:

// Highlight a specific section EnhancedTOC.highlightSection('section-id'); // Clear all highlighting EnhancedTOC.clearHighlighting(); // Reinitialize (useful for dynamic content) EnhancedTOC.reinit();

Browser Support

The enhanced TOC works in all modern browsers that support:

  • Intersection Observer API (Chrome 51+, Firefox 55+, Safari 12.1+)
  • CSS Custom Properties (all modern browsers)
  • ES6+ JavaScript features (all modern browsers)
Progressive Enhancement: The TOC gracefully degrades in older browsers, falling back to standard anchor link behavior.
