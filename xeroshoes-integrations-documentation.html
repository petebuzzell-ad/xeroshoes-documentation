Integration Overview

XeroShoes.com integrates with 36 specialized apps and services, with evidence of active implementation found in theme code, configuration files, and custom styling. This documentation is based on actual codebase analysis of the US instance only.

Data Source: Documentation based on theme export analysis, settings configuration, and code evidence. US instance (October 20, 2025) and EU instance (October 21, 2025) analyzed separately.

US Instance - Verified Active Integrations (10 Apps)

Based on codebase analysis, these integrations show evidence of active implementation:

  • Klaviyo Email Marketing: Custom CSS styling, form integration, and theme-specific configurations
  • Stamped Reviews: Custom CSS, JavaScript integration, and product page implementation
  • Searchanise Search: App blocks configured in theme settings (currently disabled)
  • Elevar Conversion Tracking: Data layer implementation in theme
  • Microsoft Clarity: Analytics tracking integration
  • Consentmo GDPR: Cookie consent management
  • Simple Bundles & Kits: Product bundling functionality with custom settings
  • Redirect Hero: URL management with live redirect capabilities
  • Orbe Geolocation: Geographic targeting and redirection
  • Klarna On-Site Messaging: Custom styling and placement integration

EU Instance - Verified Active Integrations (4 Apps)

Analysis of EU theme export (October 21, 2025) reveals limited app integration:

  • Klaviyo Email Marketing: Same app block ID as US, custom CSS styling for forms
  • Stamped Reviews: Extensive custom CSS and JavaScript integration, product page implementation
  • Redirect Hero: Both app and live redirect blocks configured (same as US)
  • Microsoft Clarity: Analytics tracking integration (same as US)
EU Instance Limitations: Missing 6 app integrations compared to US instance: Searchanise, Elevar, Consentmo GDPR, Simple Bundles, Orbe Geolocation, and Klarna On-Site Messaging.

Troubleshooting Based on Implementation Evidence

Common issues and solutions based on actual theme integration evidence.

Klaviyo Integration Issues

Evidence-Based Troubleshooting: Based on custom CSS and theme integration

  • Form Styling Issues:
    • Check CSS specificity conflicts with custom Klaviyo styles
    • Verify XeroShoes color variables are properly defined
    • Test form validation states and error message display
  • Theme-Specific Styling:
    • Verify "Born to Run" and "Xero Circle" page variants are properly styled
    • Check footer integration and responsive design
    • Test form submission and success states

Stamped Reviews Integration Issues

Evidence-Based Troubleshooting: Based on custom CSS and JavaScript integration

  • Review Display Issues:
    • Check if StampedFn is properly loaded before calling loadBadges()
    • Verify custom CSS overrides are not conflicting with Stamped's default styles
    • Test review form submission and star rating display
  • JavaScript Integration:
    • Verify StampedFn?.loadBadges() is called in all relevant components
    • Check for JavaScript errors in browser console
    • Test review badge loading in search, quickshop, and product components

App Block Configuration Issues

Evidence-Based Troubleshooting: Based on theme settings configuration

  • Disabled Apps:
    • Searchanise is currently disabled - check if it should be enabled
    • Verify app block IDs match between settings and app store
  • Configuration Settings:
    • Check Simple Bundles settings for placeholder text and option handling
    • Verify Orbe Geolocation modal text is appropriate for XeroShoes branding
    • Test Redirect Hero live redirect functionality

Apps with No Theme Evidence

Note: Many apps listed in the data export show no evidence of theme integration. These may be:

  • Configured at the Shopify admin level only
  • Using their own interfaces without theme modifications
  • Backend-only integrations (analytics, data warehouse, etc.)
  • Apps that were installed but not actively used
