Theme Architecture Overview

The XeroShoes eCommerce platform is built on a heavily customized version of the Fluid Framework theme (v3.0.0) by Praella Agency. The theme has been extensively modified to support XeroShoes' unique barefoot/minimalist shoe business model with custom sections, templates, and functionality.

Theme Details: Fluid Framework v3.0.0 by Praella Agency, extensively customized for XeroShoes' barefoot shoe business model.

Base Theme: Fluid Framework

The foundation is the Fluid Framework theme, which provides:

  • Modern, responsive design architecture
  • Advanced section and block system
  • Built-in performance optimizations
  • Extensive customization options through settings schema

Theme Configuration

The theme includes comprehensive settings for:

  • Color System: Base colors, main colors, accent colors, and gradient options
  • Typography: Font families, sizes, weights, and spacing
  • Layout: Container widths, spacing, and responsive breakpoints
  • Product Display: Grid layouts, image ratios, and product card styling

Custom Sections

XeroShoes has developed numerous custom sections to support their specific business needs and marketing requirements.

Xero Custom Landing

The flagship custom section for barefoot shoe marketing, featuring:

  • Hero section with brand messaging ("Unlock Your Natural Power™")
  • Value proposition sections explaining barefoot benefits
  • Product showcase with shoe finder integration
  • Educational content about foot health and movement
  • Custom styling with XeroShoes brand colors (#53CAEC, #F3F0EA)

Key Customization Areas

CSS & Styling

The theme includes extensive custom CSS files for specific functionality:

  • Product-specific styles: product-comfort.css, product-recently-viewed.js
  • Template styles: template-*.css for specific page types
  • Component styles: Component-specific styling for cart, search, etc.
  • Global styles: global.css, utilities.css, spacing.css

JavaScript Modules

Custom JavaScript functionality includes:

  • Product functionality: variant selection, image zoom, quickshop
  • Cart enhancements: cart form, discount handling, upsell
  • Search features: predictive search, search modal
  • Performance: lazy loading, optimization scripts

Custom Metafields

Extensive metafield system for product data:

  • Product attributes: color, activity, gender, collection
  • Marketing data: mega menu images, hover images, display titles
  • Content management: product highlights, descriptions, FAQs
  • Integration data: Google Shopping, reviews, bundles

Branding & Design System

XeroShoes maintains a consistent brand identity throughout the theme:

Color Palette

  • Primary Blue: #53CAEC (brand accent)
  • Background: #F3F0EA (warm neutral)
  • Text: #0F172B (dark navy)
  • Accent: Various activity-specific colors

Typography

  • Headings: Paralucent (custom font family)
  • Body Text: Inter (web-safe fallback)
  • Special Use: Custom font weights and styles

Performance Optimizations

The theme includes several performance optimization features:

  • Lazy Loading: lozad.js for image optimization
  • Preloading: Critical asset preloading
  • Bundle Optimization: Custom bundling for product variants
  • Speed Scripts: Performance optimization scripts
  • Image Optimization: Responsive images and WebP support

├── assets/

├── templates/

Key File Relationships

  • Landing Pages: sections/xero-custom-landing.liquid + snippets/xero-landing-hero.liquid
