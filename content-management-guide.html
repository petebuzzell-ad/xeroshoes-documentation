Strategic Content Management

This guide enables business stakeholders to maximize platform value through strategic content management across XeroShoes' dual-instance architecture. Master these procedures to drive conversion rates, improve customer experience, and accelerate business growth.

Business Impact: Effective content management directly drives conversion rates, customer engagement, and revenue growth. This guide ensures consistent, high-quality content delivery across both markets.
Critical: XeroShoes operates separate Shopify instances due to legal requirements. Content must be managed independently on both XeroShoes.com (US) and XeroShoes.eu (EU) instances to maintain compliance and market-specific optimization.

Getting Started

Accessing the Admin

XeroShoes operates two separate Shopify instances, each with its own admin interface:

US Instance (XeroShoes.com)

  1. Navigate to admin.shopify.com in your web browser
  2. Enter your email address and password
  3. Select "XeroShoes.com" from the store list
  4. You'll be taken to the US instance admin dashboard

EU Instance (XeroShoes.eu)

  1. Navigate to admin.shopify.com in your web browser
  2. Enter your email address and password
  3. Select "XeroShoes.eu" from the store list
  4. You'll be taken to the EU instance admin dashboard
Security: Always log out when finished and never share your login credentials. You may have different permissions on each instance.

User Permissions

Your account has specific permissions that determine what you can access and modify:

  • Full Access: Can manage all aspects of the store
  • Content Manager: Can edit products, pages, and blog posts
  • Order Manager: Can process orders and manage customer service
  • Analytics Viewer: Can view reports and analytics

Product Management

Adding New Products

Step-by-Step Process

  1. Navigate to Products: Click "Products" in the left sidebar
  2. Add Product: Click the "Add product" button
  3. Product Title: Enter a clear, descriptive product name
  4. Description: Write detailed product description including key features
  5. Product Type: Select appropriate category (e.g., "Barefoot Shoes")
  6. Vendor: Enter "XeroShoes"
  7. Tags: Add relevant tags for search and filtering
  8. Images: Upload high-quality product images (minimum 3-5 images)
  9. Variants: Set up size and color options
  10. Pricing: Enter price and compare-at price if applicable
  11. Inventory: Set initial stock quantities
  12. SEO: Add page title and meta description
  13. Save: Click "Save" to create the product

Product Information Best Practices

  • Use clear, descriptive product titles
  • Include key features and benefits in descriptions
  • Add activity-specific tags (running, hiking, casual)
  • Include sizing information and fit notes
  • Highlight unique barefoot benefits

Managing Product Images

Image Requirements

  • Format: JPG or PNG files
  • Size: Minimum 1024x1024 pixels
  • Quality: High resolution, well-lit photos
  • Quantity: 3-5 images per product minimum

Image Types

  • Hero Image: Main product shot on white background
  • Lifestyle Images: Product in use or on person
  • Detail Shots: Close-ups of key features
  • Size Comparison: Show product next to common objects

Uploading Images

  1. In the product edit page, scroll to "Media" section
  2. Click "Add files" or drag and drop images
  3. Arrange images in desired order (first image is the main product image)
  4. Add alt text for accessibility
  5. Save changes

Troubleshooting

Common Issues

Images Not Uploading

  • Check file size (should be under 20MB)
  • Verify file format (JPG, PNG, or WebP)
  • Try uploading one image at a time
  • Clear browser cache and try again

Changes Not Appearing

  • Clear browser cache and refresh page
  • Check if changes were saved properly
  • Verify you're looking at the correct page
  • Wait a few minutes for changes to propagate

Permission Issues

  • Contact administrator for permission changes
  • Verify you're logged into the correct account
  • Check if you have the right role assigned
