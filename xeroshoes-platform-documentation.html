Executive Summary

Platform Overview

The XeroShoes eCommerce platform operates on a dual-instance Shopify Plus architecture, serving the US market through XeroShoes.com and the EU market through XeroShoes.eu. This separation is required due to legal and regulatory compliance requirements, ensuring proper business entity structure and data sovereignty.

Platform Status: Both instances are fully operational with 99.9% uptime, serving customers across North America and Europe with localized experiences and compliance requirements.

Content Inventory

  • Products: 388 products with comprehensive metafield data
  • Collections: 223 collections (209 smart collections, 14 custom collections)
  • Pages: 97 pages across both instances
  • Blog Posts: 445 blog posts (US instance)
  • Redirects: 3,455 redirects for SEO preservation
  • Metaobjects: 500+ metaobjects for structured content

Key Technical Components

  • Theme: Fluid Framework v3.0.0 (Praella Agency)
  • Custom Sections: 78+ custom sections including Xero-specific landing pages
  • Metafields: Extensive custom metafield system for product data
  • Integrations: 15+ third-party integrations for payments, shipping, and marketing
  • Multi-Region: Separate Shopify Plus instances for US and EU compliance

Content Management Workflows

Standardized procedures for managing content across both XeroShoes instances.

Page Creation Workflow

  1. Access Shopify Admin → Online Store → Pages
  2. Click "Add page" to create new page
  3. Enter page title and URL handle
  4. Select appropriate template from available options
  5. Add content using rich text editor
  6. Configure SEO settings (title, description, keywords)
  7. Set visibility and publication date
  8. Save and publish page
  9. Repeat process on both US and EU instances

Blog Post Management

  1. Access Online Store → Blog posts
  2. Create new blog post with title and content
  3. Add featured image and excerpt
  4. Set publication date and author
  5. Add relevant tags for categorization
  6. Configure SEO metadata
  7. Publish and verify on live site

Product Management Workflows

Standardized procedures for product creation, updates, and management across both instances.

New Product Creation

  1. Access Products → Add product
  2. Enter basic product information (title, description, vendor)
  3. Set product type and add relevant tags
  4. Upload product images (minimum 3-5 images)
  5. Configure variants (sizes, colors, styles)
  6. Set pricing and inventory quantities
  7. Add custom metafields for detailed attributes
  8. Configure SEO settings
  9. Save and publish product
  10. Repeat process on both US and EU instances

Product Update Workflow

  1. Identify product requiring updates
  2. Access product in Shopify Admin
  3. Update relevant information (pricing, inventory, descriptions)
  4. Update metafields if necessary
  5. Preview changes before publishing
  6. Save and publish updates
  7. Apply same changes to corresponding instance
  8. Verify changes on live site

Operations & Runbooks

User & Access Management

User access provisioning and permission management across both instances.

User Roles & Permissions

  • Store Owner: Full access to all features and settings
  • Admin: Full access except billing and user management
  • Content Manager: Products, pages, blog posts, collections
  • Order Manager: Orders, customers, fulfillment
  • Analytics Viewer: Reports and analytics only

Access Provisioning Process

  1. Determine required access level based on role
  2. Create user account in Shopify Admin
  3. Assign appropriate permissions
  4. Send invitation email to user
  5. User completes account setup
  6. Verify access and permissions
  7. Repeat process for both instances if needed

Troubleshooting Guides

Common issues and resolution procedures for the XeroShoes platform.

Common Issues

  • Images Not Loading: Check file size and format, clear cache
  • Payment Processing Errors: Verify payment gateway configuration
  • Inventory Sync Issues: Check metafield configurations
  • Email Delivery Problems: Verify Klaviyo integration settings

Emergency Procedures

  • Site Down: Contact Shopify support immediately
  • Payment Issues: Switch to backup payment method
  • Data Loss: Restore from Matrixify backup
  • Security Breach: Follow incident response protocol
